## 💡 CÓMO FUNCIONAN

### Cliente (Frontend)
1. Usuario pega texto en el formulario
2. Hace click en "Procesar"
3. JavaScript valida y envía al servidor
4. Muestra indicador de carga
5. Recibe respuesta y muestra resultados
6. Usuario puede copiar cada sección

### Servidor (Backend)
1. Recibe el texto del cliente
2. Valida que tenga 10-10,000 caracteres
3. Genera prompts para IA
4. Llama a API de Groq/Ollama
5. Parsea la respuesta
6. Devuelve JSON con resultados
