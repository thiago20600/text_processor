## 🎯 EJEMPLOS DE USO

### Ejemplo 1: Procesar un artículo de blog

**Texto de entrada:**
```
La inteligencia artificial ha revolucionado múltiples industrias en los últimos años. 
Desde el reconocimiento de imágenes hasta la traducción automática, la IA está transformando 
cómo trabajamos y vivimos. Los algoritmos de aprendizaje profundo pueden analizar millones 
de datos en segundos, identificar patrones que los humanos nunca podrían encontrar.

Sin embargo, la IA también presenta desafíos importantes. Preocupaciones sobre privacidad, 
sesgos algorítmicos y desplazamiento laboral son cuestiones críticas que la sociedad debe 
abordar. Es fundamental que el desarrollo de la IA sea responsable y ético.
```

**Resultados generados:**

📝 **Resumen:**
```
La inteligencia artificial ha transformado múltiples industrias mediante algoritmos 
de aprendizaje profundo capaces de procesar grandes volúmenes de datos. Aunque ofrece 
capacidades revolucionarias, presenta desafíos importantes relacionados con privacidad, 
sesgos algorítmicos y empleo que requieren desarrollo ético y responsable.
```

⭐ **Puntos Clave:**
```
- IA revoluciona múltiples industrias (imágenes, traducción)
- Algoritmos de aprendizaje profundo analizan millones de datos
- Identifica patrones indetectables para humanos
- Privacidad como preocupación crítica
- Sesgos algorítmicos requieren abordaje responsable
- Desplazamiento laboral es una realidad
- Desarrollo ético de IA es fundamental
```

❓ **Preguntas de Examen:**
```
1. ¿Cuál es una aplicación de la IA mencionada?
   a) Manufactura manual
   b) Traducción automática
   c) Agricultura tradicional
   d) Todas las anteriores
   ✓ Respuesta: b)

2. ¿Cuál es el principal desafío mencionado sobre IA?
   a) Costo de hardware
   b) Falta de datos
   c) Sesgos algorítmicos y privacidad
   d) Ninguno
   ✓ Respuesta: c)

3. ¿Qué pueden hacer los algoritmos de aprendizaje profundo?
   a) Procesar millones de datos en segundos
   b) Identificar patrones complejos
   c) Ambas
   d) Ninguna
   ✓ Respuesta: c)

4. ¿Por qué es importante el desarrollo ético de la IA?
   a) Porque es obligatorio por ley
   b) Porque afecta privacidad, empleo y sesgo
   c) Porque la IA es perfecta
   d) Sin razón especial
   ✓ Respuesta: b)

5. ¿Qué es lo que los algoritmos de IA pueden identificar?
   a) Solo objetos visibles
   b) Patrones indetectables para humanos
   c) Solo números
   d) Nada
   ✓ Respuesta: b)
```

### Ejemplo 2: Procesar un artículo científico

**Texto de entrada:**
```
El cambio climático es causado principalmente por las emisiones de gases de efecto invernadero, 
particularmente CO2. El dióxido de carbono se produce por la quema de combustibles fósiles como 
petróleo, carbón y gas natural. Desde la Revolución Industrial, las concentraciones de CO2 en 
la atmósfera han aumentado más del 40%.

Los principales efectos incluyen el calentamiento global, aumento del nivel del mar, cambios 
en los patrones climáticos y extinción de especies. Para mitigarlo, es necesario reducir 
emisiones, desarrollar energías renovables y reforestar. Los acuerdos internacionales como 
el Acuerdo de París buscan limitar el calentamiento a 1.5 grados Celsius.
```

**Resultados esperados:**

📝 **Resumen:**
```
El cambio climático resulta primordialmente de emisiones de gases de efecto invernadero, 
especialmente CO2 generado por combustibles fósiles, cuyas concentraciones han aumentado 40% 
desde la Revolución Industrial. Sus efectos incluyen calentamiento global, alza del nivel marino 
y extinción de especies, requiriendo estrategias de mitigación como energías renovables y 
reforestación, conforme acuerdos internacionales como el Acuerdo de París.
```

⭐ **Puntos Clave:**
```
- Gases de efecto invernadero son causa principal del cambio climático
- CO2 proviene de quema de combustibles fósiles
- Aumento de 40% de CO2 desde Revolución Industrial
- Efectos: calentamiento global, alza del nivel del mar
- Amenaza de extinción de especies
- Soluciones: reducir emisiones y energías renovables
- Acuerdo de París limita calentamiento a 1.5°C
```

### Ejemplo 3: Procesar un resumen de libro

**Texto de entrada:**
```
"1984" de George Orwell es una novela distópica que presenta un mundo totalitario donde 
el Partido controla todos los aspectos de la vida humana. El protagonista, Winston Smith, 
trabaja en el Ministerio de la Verdad alterando registros históricos. 

El gobierno usa vigilancia constante, propaganda y lenguaje manipulado (Neolengua) para 
controlar el pensamiento. Winston intenta rebelarse pero es descubierto, torturado y 
finalmente convertido a las creencias del Partido. 

La novela advierte sobre los peligros del totalitarismo, la pérdida de libertad y 
la manipulación de la verdad por parte del poder.
```

**Resultados esperados:**

⭐ **Puntos Clave:**
```
- Novela distópica sobre un régimen totalitario
- El Partido controla todos los aspectos de la vida
- Winston Smith intenta rebelarse contra el sistema
- Vigilancia constante y propaganda son herramientas de control
- Neolengua es utilizada para manipular el pensamiento
- Represión y tortura son castigos para disidentes
- La novela advierte sobre peligros del totalitarismo
```

❓ **Preguntas de Examen:**
```
1. ¿Cuál es el ocupación de Winston Smith?
   a) Militar
   b) Trabajador en Ministerio de la Verdad
   c) Profesor
   d) Periodista
   ✓ Respuesta: b)

2. ¿Qué es la "Neolengua"?
   a) Un idioma extranjero
   b) Un lenguaje manipulado para controlar el pensamiento
   c) La lengua de la rebeldía
   d) Un dialecto antiguo
   ✓ Respuesta: b)
```

## 🔧 Cómo usar la aplicación paso a paso

### 1. Abre la aplicación

```
http://localhost:8000
```

### 2. Pega tu texto

En el área de texto grande, pega el contenido que deseas procesar:

![Interface]
┌────────────────────────────────────┐
│ 📚 AI Text Processor               │
│                                    │
│ Ingresa tu texto                   │
│ ┌──────────────────────────────┐   │
│ │ [Pega aquí tu texto...]      │   │
│ │                              │   │
│ │                              │   │
│ └──────────────────────────────┘   │
│ 0 / 10000 caracteres               │
│                                    │
│ [🚀 Procesar texto]                │
└────────────────────────────────────┘

### 3. Haz clic en "Procesar texto"

La aplicación muestra un indicador de carga mientras procesa:

```
Procesando tu texto con IA...
Esto puede tardar unos segundos...
```

### 4. Visualiza los resultados

Se mostran tres secciones con los resultados:

```
┌─ 📝 Resumen ────────────────────┐
│ [Copiar]                        │
│ [Texto del resumen aquí...]     │
└─────────────────────────────────┘

┌─ ⭐ Puntos Clave ───────────────┐
│ [Copiar]                        │
│ - Punto 1                       │
│ - Punto 2                       │
│ - Punto 3                       │
└─────────────────────────────────┘

┌─ ❓ Preguntas de Examen ────────┐
│ [Copiar]                        │
│ 1. Pregunta...                  │
│    a) Opción a                  │
│    b) Opción b                  │
│    c) Opción c                  │
│    d) Opción d                  │
│    ✓ Respuesta: a)              │
└─────────────────────────────────┘

[➕ Procesar otro texto]
```

### 5. Copia los resultados

Cada sección tiene un botón "Copiar" para copiar al portapapeles.

### 6. Procesa otro texto (opcional)

Haz clic en "Procesar otro texto" para limpiar y procesar un nuevo documento.

## 🎨 Comandos útiles

### Para Windows:
```bash
# Activar entorno
venv\Scripts\activate

# Ejecutar aplicación
python main.py

# Ver en navegador
start http://localhost:8000

# Instalar dependencias
pip install -r requirements.txt
```

### Para macOS/Linux:
```bash
# Activar entorno
source venv/bin/activate

# Ejecutar aplicación
python main.py

# Ver en navegador
open http://localhost:8000

# Instalar dependencias
pip install -r requirements.txt
```

## 🔌 API REST (para integraciones)

### Endpoint: POST /api/process

**Request:**
```bash
curl -X POST http://localhost:8000/api/process \
  -H "Content-Type: application/json" \
  -d '{"text":"Tu texto aquí con más de 10 caracteres"}'
```

**Response exitoso:**
```json
{
  "success": true,
  "summary": "Resumen del texto...",
  "key_points": "Puntos clave:\n- Punto 1\n- Punto 2",
  "questions": "Preguntas:\n1. ¿Pregunta?\n   a) Opción\n   ✓ b)"
}
```

**Response error:**
```json
{
  "success": false,
  "error": "Por favor ingresa un texto con al menos 10 caracteres"
}
```

## 💡 Tips de uso

✅ **Textos que funcionan bien:**
- Artículos de blog y noticias
- Resúmenes de libros
- Apuntes de clase
- Papers y artículos científicos
- Documentación técnica

⚠️ **Textos que pueden ser problemáticos:**
- Muy cortos (menos de 50 caracteres)
- En idiomas muy diferentes (mejor en inglés/español)
- Con mucho código (no es texto narrativo)
- Tablas de datos sin contexto
